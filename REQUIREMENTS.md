# VoxCPM 在线配音工具 — 需求规格说明书

## 1. 项目概述

**项目名称**：VoxCPM Studio — AI 多语言配音工坊

**一句话描述**：网页端输入文字、选声音，一键生成多语言语音并在线播放/下载。

**核心价值**：让用户零门槛使用 VoxCPM 2 的高质量语音合成能力，无需安装 Python、下载模型。

## 2. 目标用户

| 用户类型 | 场景 | 核心需求 |
|----------|------|----------|
| 视频创作者 | 为短视频配音 | 多语言、自然语音、快速生成 |
| 内容运营 | 文章转音频 | 长文本、批量合成 |
| 开发者 | 测试声音克隆效果 | 上传参考音频，快速试听 |

## 3. 功能需求

### 3.1 P0 — MVP 必须有的功能

**F1 — 文本转语音**
- 用户输入文本（支持中英日韩等多语言）
- 选择语速、情感参数
- 点击生成，返回音频
- 在线播放 + 下载为 WAV

**F2 — 声音克隆**
- 用户上传一段 5-30 秒的参考音频
- 系统用该音色合成目标文本
- 播放 + 下载

**F3 — 预设音色库**
- 5-10 个内置声音模板（男/女/中/英）
- 用户无需上传即可直接选用

**F4 — 合成历史**
- 最近 20 条合成记录
- 可重新播放、下载、删除

### 3.2 P1 — 后续迭代

**F5 — 批量合成**
- 上传文本文件（每行一句），批量生成

**F6 — 参数调优面板**
- 语速 (speed)、情感强度 (cfg_value)、采样步数 (timesteps)
- 实时预览小段

**F7 — 用户系统**
- 注册/登录、配额限制

### 3.3 不做的

- 不做实时流式语音（延迟太高）
- 不做音乐生成
- 不做语音识别（ASR，只用 VoxCPM 自带的 SenseVoice 做 prompt 转写）

## 4. 非功能需求

| 需求 | 指标 |
|------|------|
| 响应时间 | 10 字文本 < 3 秒，50 字 < 10 秒 |
| 并发 | 支持 3 个用户同时合成（GPU 排队） |
| 可用性 | 99% uptime（单机部署） |
| 文件保留 | 生成的音频保留 24 小时后自动清理 |
| 浏览器兼容 | Chrome / Edge / Safari 最新版，移动端响应式 |

## 5. 技术约束

- 模型：VoxCPM 2 (`openbmb/VoxCPM2`)
- 推理：需要 NVIDIA GPU（最低 8GB 显存，推荐 T4/A10）
- 后端：Python + FastAPI
- 前端：Vue 3 SPA
- 部署：Docker + docker-compose
- 音频格式：WAV 24kHz
